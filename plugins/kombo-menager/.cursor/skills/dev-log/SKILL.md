# Dev Log — kombo-menager

## On every completed task, append to docs/DEVLOG.md:

### Format:

## [{version}] — {date} — Ilija / s7codedesign

- {action}: {description}

### Action types:

- Added: new feature or file
- Changed: modification to existing code
- Fixed: bug fix
- Removed: deleted code or file
- Phase: phase completion marker

### Rules:

- Always append, never overwrite previous entries
- Date format: DD.MM.YYYY
- Version bumps only on phase completion
- One line per change, keep it concise
